## Мета роботи
Створити графічний інтерфейс користувача та реалізувати **CRUD-операції** (створення, перегляд, редагування, видалення об’єктів) у програмному застосунку, забезпечивши взаємодію UI з логікою програми.

## Завдання роботи
1. Створити графічний інтерфейс користувача програмного застосунку.  
2. Реалізувати створення нових об’єктів.  
3. Реалізувати перегляд списків об’єктів.  
4. Реалізувати редагування та видалення об’єктів.  
5. Забезпечити взаємодію UI з логікою застосунку.

## Використані технології та підходи
- **Мова програмування:** C#  
- **UI:** Windows Forms (`Form1`)  
- **Колекції:** `List<T>`, `Dictionary<TKey,TValue>`  
- **CRUD-операції:** Додавання, перегляд, редагування, видалення об’єктів  
- **Подієво-орієнтоване програмування:** Обробка подій кнопок, форм та елементів управління  
- **Взаємодія UI та логіки програми:** Розділення бізнес-логіки та графічного інтерфейсу  
- **SOLID принципи:** Забезпечення структурованості та підтримуваності коду  

## Опис реалізованих CRUD-операцій
- **Create (Створення):** Додавання нових об’єктів через форму введення даних.  
- **Read (Перегляд):** Відображення списку об’єктів у елементах `ListBox` або `DataGridView`.  
- **Update (Редагування):** Зміна властивостей об’єктів через форму редагування.  
- **Delete (Видалення):** Видалення об’єктів зі списку та колекцій через кнопку або контекстне меню.
